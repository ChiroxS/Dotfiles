## Shell setup
- Terminal - iTerm2
- Shell - zsh
- Shell framework - ohmyzsh
- Shell theme - powerlevel10k
- Package manager - Brew
- Tools: exa(ls), bat(cat)

## Neovim setup 
- Base config - NvChad
- Syntax highlighting - nvim-treesitter 
- Language servers - gopls (Go), rust-analyzer (Rust), sourcekit (Swift) 
- Formatting - conform.nvim
- Linting - lint.nvim
- Plugin manager - lazy.nvim
- Package manager - mason.nvim 
- Files & search - telescope.nvim (fzf, livegrep)
- Theme - bearded_arc
- Dashboard - dashboard.nvim
